# Kakaosec Coding Test

### Local Setting
```json
{
  "prettier.configPath": "",
  "editor.formatOnSave": false,
  "editor.defaultFormatter": "esbenp.prettier-vscode",
  "editor.codeActionsOnSave": [
    "source.fixAll.eslint",
    "source.fixAll.stylelint"
  ]
}
```
